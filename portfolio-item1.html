Project Overview

This project's goal was to display a real-world website that seeks to display the gist of how any restuarant website looks. The original website was outdated and outsourced their ordering system to a third-party vendor. Many times, this requires a hefty fee to the third-party vendor. So I re-create the website to make it look more visually-appealing and allow it to look like an (almost) fully, functional website without the use of a third-party ordering system.

As with anything in software, there is never a perfect software, website, or application. There is always room for improvement and I seek to identify those key aspects in all of my projects. For this project, here's a few things that it will NOT do(but will be worked on):

  • Save added items to cart on refresh
  • Sign up an account
  • Save favorites
  • Place orders hypothetically
